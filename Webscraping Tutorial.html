Webscraping Tutorial

How to get started with Webscraping + EXAMPLES


 Note: This documentation is meant for a Windows 10 machine with python preinstalled. Quotations are not included when typing in terminal and parentheses in the quotations refer to the unique names that the user is supposed to enter.

First, you would want to add a directory where you would save your Scrapy projects and begin with adding a Python virtual environment. This way the python packages you add for this project will not conflict with your system environment. To do this, open up the terminal by searching up the command prompt in the windows start screen and type “cd..” ENTER until the directory on the left hand side is just the disk drive. Then type “python -m venv (your scrapy project directory)” ENTER. Use the image above as an example. Then type “pip install scrapy” ENTER.

Then to start the actual scrapy project in the directory by searching back to the directory by typing “cd (folder name)” as done above, but with the names of the folders that you described. Once you are in the desired directory type “scrapy startproject (project_name)” ENTER. After letting the code run, if you check your project in file explorer you will see some newly added files. 

Route to the project name folder and open the “spiders” folder and add a new python file. For our documentation we will make a scraper that will extract a few quotes from a target website. It is very important to not include any spaces in the name of the file and to remember this name as this is the name of the “spider” that will “crawl” over the website and extract the information. 


Open the python file in any python editor and this is where you will write the specific tasks that your spider will complete. Keeping the imports the same create a new class in the same format above. For now you can use the same code as us for the parse function but make sure the name of the spider is the same as the file name and the urls are any website that you would like to scrape. 



Go back into terminal and and go to the directory where the scrapy project name is using the same “cd (directory name)” steps done above. Then type “scrapy crawl (the name that you typed in the python file in the previous step)” ENTER. 


Then if you go back to the directory in file explorer you will see a browser executable file for every URL you added for your scraper. 




Then if you double click on the file and open it in the browser of your choice you will see all your data that you specified for your spider to find all neatly arranged in this HTML document. You have successfully scraped your first website! Now there are countless more possibilities and so if you need any more help make sure to check out the scrapy documentation that was linked on our website.
